BitSynth

a free app for Windows Phone

DOWNLOAD THE APP

BitSynth is an synthesizer app for Windows Phone, built by 8-bit musician Stenobot. Like most of Stenobot's music, the synth utilizes "8-bit" bleeps and bloops produced by old videogame systems--particularly the Game Boy. Use BitSynth to cover your favorite videogame soundtracks or to write your own music. And don't forget to try the TILT/SHAKE function, which lets you use the phone's accelerometer to do some crazy stuff!

Feature What it does
TILT/SHAKE Tilt the phone back and forth to slowly (or quickly!) change the pitch, or shake the phone while holding one or more keys for some crazy, glitched out effects!
RELEASE Switch between a slow release (sound fades out) and no release (sound plays as long as you're holding the key.
VOICE Switch between 4 different sound waves.
OCTAVE Change the keyboard octave up or down on the fly!
